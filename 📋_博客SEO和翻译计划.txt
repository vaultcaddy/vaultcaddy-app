🎯 SEO 優化清單
=================================================================

### 每篇文章需要優化的 SEO 元素:

1. **Meta Title (標題)**
   - 長度: 50-60 字符
   - 包含主要關鍵詞
   - 吸引點擊
   - 格式: [主題] | VaultCaddy

2. **Meta Description (描述)**
   - 長度: 150-160 字符
   - 包含次要關鍵詞
   - 清晰的價值主張
   - 包含行動呼籲

3. **Meta Keywords (關鍵詞)**
   - 5-10 個相關關鍵詞
   - 包含主要和長尾關鍵詞
   - 逗號分隔

4. **Open Graph Tags (社交媒體)**
   - og:title
   - og:description
   - og:image
   - og:url
   - og:type

5. **Structured Data (結構化數據)**
   - JSON-LD 格式
   - Article schema
   - Organization schema
   - BreadcrumbList schema

6. **內部鏈接**
   - 鏈接到相關文章
   - 鏈接到主要頁面
   - 自然的錨文本

7. **H1-H6 標題結構**
   - 清晰的層級
   - 包含關鍵詞
   - 易於閱讀

8. **圖片 ALT 文本**
   - 描述性
   - 包含關鍵詞
   - 易於理解

📝 SEO 優化模板
=================================================================

### 標準 Meta 標籤模板:

```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Primary Meta Tags -->
    <title>[文章標題] | VaultCaddy</title>
    <meta name="title" content="[文章標題] | VaultCaddy">
    <meta name="description" content="[150字描述]">
    <meta name="keywords" content="[關鍵詞1], [關鍵詞2], [關鍵詞3]">
    <meta name="author" content="VaultCaddy">
    
    <!-- Open Graph / Facebook -->
    <meta property="og:type" content="article">
    <meta property="og:url" content="https://vaultcaddy.com/blog/[文章名].html">
    <meta property="og:title" content="[文章標題] | VaultCaddy">
    <meta property="og:description" content="[150字描述]">
    <meta property="og:image" content="https://vaultcaddy.com/images/blog/[文章圖片].jpg">
    
    <!-- Twitter -->
    <meta property="twitter:card" content="summary_large_image">
    <meta property="twitter:url" content="https://vaultcaddy.com/blog/[文章名].html">
    <meta property="twitter:title" content="[文章標題] | VaultCaddy">
    <meta property="twitter:description" content="[150字描述]">
    <meta property="twitter:image" content="https://vaultcaddy.com/images/blog/[文章圖片].jpg">
    
    <!-- Canonical URL -->
    <link rel="canonical" href="https://vaultcaddy.com/blog/[文章名].html">
    
    <!-- Structured Data -->
    <script type="application/ld+json">
    {
      "@context": "https://schema.org",
      "@type": "Article",
      "headline": "[文章標題]",
      "description": "[150字描述]",
      "author": {
        "@type": "Organization",
        "name": "VaultCaddy"
      },
      "publisher": {
        "@type": "Organization",
        "name": "VaultCaddy",
        "logo": {
          "@type": "ImageObject",
          "url": "https://vaultcaddy.com/logo.png"
        }
      },
      "datePublished": "2025-12-05",
      "dateModified": "2025-12-05"
    }
    </script>
</head>
```
