# 如果选择方案B（图片在根目录），需要修改 index.html

## 修改内容

找到这两行：
```
<meta property="og:image" content="https://vaultcaddy.com/images/og-vaultcaddy-main.jpg">
<meta name="twitter:image" content="https://vaultcaddy.com/images/twitter-vaultcaddy-main.jpg">
```

改为：
```
<meta property="og:image" content="https://vaultcaddy.com/og-image.jpg">
<meta name="twitter:image" content="https://vaultcaddy.com/og-image.jpg">
```

## 操作步骤

1. 打开 index.html
2. 使用查找功能 (Ctrl+F / Cmd+F)
3. 搜索：`og-vaultcaddy-main.jpg`
4. 替换为：`og-image.jpg`
5. 保存文件
