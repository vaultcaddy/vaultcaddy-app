## 🧪 A/B测试执行指南

### 步骤1：安装Google Optimize

```html
<!-- 在head标签中添加 -->
<script src="https://www.googleoptimize.com/optimize.js?id=OPT-XXXXXXX"></script>
```

### 步骤2：创建实验

1. 登入 https://optimize.google.com/
2. 创建新实验 → 选择"A/B测试"
3. 输入实验名称和测试页面URL
4. 创建变体（变体A、变体B）

### 步骤3：定义目标

**主要目标**：注册转化（Google Analytics事件）
- 事件类别：User
- 事件操作：Registration
- 事件标签：Completed

**次要目标**：
- CTA按钮点击率
- 页面停留时间 > 2分钟
- 跳出率 < 40%

### 步骤4：流量分配

- Control（原版）：50%
- Variant A：25%
- Variant B：25%

### 步骤5：测试周期

- **最少运行时间**：2周
- **最少样本量**：1,000访客/变体
- **置信度要求**：95%
- **预期提升**：>15%

### 步骤6：分析结果

```
如果变体A的转化率显著高于Control：
1. 停止实验
2. 将变体A设为默认版本
3. 记录提升百分比
4. 开始下一个测试

如果没有显著差异：
1. 继续运行测试（最多4周）
2. 如果仍无差异，放弃该变体
3. 尝试更激进的变化
```
