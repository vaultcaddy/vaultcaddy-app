## ⚠️ 注意事项

### Qwen-OCR 限制

根据文档，需要注意：

1. **不支持System Message**: 
   - 所有指令必须通过User Message传入
   - 需要在提示词中包含所有要求

2. **无多轮对话能力**:
   - 每次调用都是独立的
   - 不能基于之前的对话上下文

3. **幻觉风险**:
   - 文字太小或分辨率低时可能出现幻觉
   - 建议图像预处理确保清晰

4. **PDF处理**:
   - 需要先将PDF转换为图像序列
   - 使用多图像输入方式处理多页文档

### 实施建议

```javascript
// 处理多页PDF
async function processPDF(pdfFile) {
    // 1. 将PDF转换为图像序列
    const images = await pdfToImages(pdfFile);
    
    // 2. 使用多图像输入
    const completion = await client.chat.completions.create({
        model: 'qwen-vl-ocr',
        messages: [
            {
                role: 'user',
                content: [
                    ...images.map(img => ({ type: 'image', image: img })),
                    { type: 'text', text: '提取所有页面的交易记录' }
                ]
            }
        ]
    });
    
    return completion.choices[0].message.content;
}
```
