### 方法1: 添加更多翻译 (推荐)

1. **编辑翻译数据库:**
```bash
open complete-translations.json
```

2. **添加页脚翻译:**
```json
{
  "index_complete": {
    "zh": {
      "footer_about": "關於我們",
      "footer_contact": "聯絡我們"
    },
    "en": {
      "footer_about": "About Us",
      "footer_contact": "Contact"
    }
  }
}
```

3. **重新运行脚本:**
```bash
python3 complete-translate.py
```

### 方法2: 手动编辑

直接编辑翻译后的文件:
```bash
# 编辑英文版
```
