## ⚠️ 常见错误和解决方法

### 错误 1：OG 图片不显示

**可能原因**：
- 图片 URL 不是 HTTPS
- 图片文件太大（> 8MB）
- 图片尺寸不对

**解决方法**：
```bash
# 检查图片 URL
curl -I https://vaultcaddy.com/images/og-preview-zh.jpg

# 应该返回：
HTTP/2 200 
content-type: image/jpeg
content-length: [< 1MB]

# 如果返回 404，检查文件路径和权限
```

### 错误 2：社交媒体缓存旧内容

**解决方法**：
```
Facebook:
1. https://developers.facebook.com/tools/debug/
2. 输入 URL
3. 点击 "Fetch new information"
4. 点击 "Scrape Again"

Twitter:
1. https://cards-dev.twitter.com/validator
2. 输入 URL
3. 点击 "Preview card"
4. 等待缓存更新（可能需要数小时）

WhatsApp:
- 等待自然过期（24-48小时）
- 或使用 URL 参数：?v=20251227
```

### 错误 3：Google 未重新抓取

**解决方法**：
```
1. 提交 sitemap：
   https://search.google.com/search-console
   → 站点地图 → 添加新的站点地图

2. 手动请求索引（每天限额约 10 个URL）

3. 等待自然抓取（1-4 周）

4. 确保 robots.txt 允许抓取：
   User-agent: *
   Allow: /
```
