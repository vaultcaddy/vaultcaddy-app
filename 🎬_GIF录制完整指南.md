### 录制步骤

**使用QuickTime**:

```bash
1. 打开QuickTime Player
2. 文件 → 新建屏幕录制
3. 点击录制按钮旁边的下拉箭头
4. 选择"内建麦克风"（如需旁白）或"无"
5. 点击红色录制按钮
6. 拖动选择录制区域（1280x800）
7. 点击"开始录制"

按照脚本操作:
  0:00 - 显示定价页面（₩7998/월）
  0:02 - 点击"파일 업로드"
  0:04 - 选择PDF
  0:06 - 显示"처리 중..."
  0:08 - 显示"완료!"
  0:10 - 显示最终结果
  0:12 - 停止录制

8. 点击停止按钮（菜单栏）
9. 保存为: chase-bank-demo-ko.mov
```

## 💡 专业提示

### 录制技巧

```
1. 保持鼠标移动缓慢流畅
2. 点击时暂停0.5秒
3. 避免快速移动（会导致模糊）
4. 确保每个场景停留足够时间
5. 最后一帧停留1秒（脚本会自动处理）
```

### 常见问题

**Q: 录制时出现闪烁？**
```
A: 关闭Chrome的硬件加速
   设置 → 高级 → 系统 → 关闭"使用硬件加速"
```

**Q: 文件太大？**
```
A: 不用担心，我的脚本会自动优化
   MOV可能100MB+，但GIF会压缩到<2MB
```

**Q: 录错了怎么办？**
```
A: 重新录制即可，覆盖文件
   录制很快（每个30分钟）
```
