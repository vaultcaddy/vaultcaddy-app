## 🎨 設計特點

### 按鈕設計
- **寬度**: 280px（從 200px 增加）
- **高度**: 自適應（padding: 0.875rem 1rem）
- **配色**:
  - 激活狀態：藍色背景（#667eea）+ 白色文字
  - 未激活狀態：白色背景 + 藍色文字 + 藍色邊框
- **圖標**: 
  - 銀行對帳單：`fa-university`
  - 發票/請求書：`fa-file-invoice`

### 上傳區域設計
- **背景**: 白色半透明（rgba(255, 255, 255, 0.95)）
- **最大寬度**: 800px
- **圓角**: 20px
- **陰影**: 0 10px 40px rgba(0,0,0,0.15)

### 拖放區域設計
- **邊框**: 3px 虛線藍色（#667eea）
- **背景**: 淺藍色（rgba(102, 126, 234, 0.05)）
- **懸停效果**: 背景變深、邊框變實

## 🔧 技術實現

### HTML 結構
```html
<div id="hero-upload-zone">
  <!-- 文檔類型選擇 -->
  <div>
    <button id="doc-type-statement" onclick="selectDocType('statement')">
      <i class="fas fa-university"></i>
      <span>銀行對帳單</span>
    </button>
    <button id="doc-type-invoice" onclick="selectDocType('invoice')">
      <i class="fas fa-file-invoice"></i>
      <span>發票</span>
    </button>
  </div>
  
  <!-- 拖放區域 -->
  <div id="dropzone">...</div>
  
  <!-- 隱藏的文件輸入 -->
  <input type="file" id="file-input">
  
  <!-- 信任標誌 -->
  <div>...</div>
  
  <!-- 促銷橫幅 -->
  <div>...</div>
</div>
```

### JavaScript 功能
- `selectDocType(type)`: 切換文檔類型（statement/invoice）
- `dropzone` 事件監聽：拖放文件
- `file-input` 事件監聽：點擊上傳
- 未登入時：暫存文件 → 彈出登入模態框
- 登入後：自動創建 "First_Project" → 跳轉並處理
