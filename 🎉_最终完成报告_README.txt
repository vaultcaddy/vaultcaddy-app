🚀 如何继续完成翻译
=================================================================

### 方法1: 使用现有工具继续AI翻译 (推荐)

**步骤:**
1. 编辑 translate-content.json，添加更多翻译内容
2. 修改 auto-translate.py，添加更多替换规则
3. 运行脚本: `python3 auto-translate.py`
4. 检查生成的文件
5. 人工校对和优化

**优点:** 快速、一致性好
**时间:** 2-3小时/语言

### 方法2: 使用ChatGPT/Claude手动翻译

**步骤:**
1. 复制index.html的一段内容
2. 发送给AI: "请将以下HTML翻译成英文，保持HTML标签不变"
3. 复制翻译结果
4. 替换到en/index.html对应位置
5. 重复以上步骤

**优点:** 质量更高、更灵活
**时间:** 4-6小时/语言

### 方法3: 雇佣专业翻译

**步骤:**
1. 导出需要翻译的文本到Excel
2. 发送给翻译人员
3. 收到翻译后，用auto-translate.py批量替换
4. 或手动替换到HTML文件中

**优点:** 最高质量、最专业
**时间:** 1-2周
**成本:** HKD $3,000-5,000/语言

**1. 语言选择器不工作**

   - 清除浏览器缓存
   - 确保language-selector.js已加载

**2. 翻译脚本报错**
   - 确保Python 3已安装
   - 检查文件路径正确
   - 确保translate-content.json格式正确

**3. 页面显示异常**
   - 检查HTML语法
   - 确保UTF-8编码
   - 验证CSS和JS文件路径

### 有用的命令:

```bash
# 查看文件编码
file -I en/index.html

# 查找未翻译的中文
```
