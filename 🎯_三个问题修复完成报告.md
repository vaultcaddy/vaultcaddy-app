## 📝 **技术细节**

### 问题1：居中修复方案

**使用Flexbox布局**：
```css
.hero-content {
    display: flex;          /* 启用Flexbox */
    flex-direction: column; /* 垂直排列 */
    align-items: center;    /* 水平居中 */
    text-align: center;     /* 文字居中 */
}
```

**优点**：
- ✅ 所有子元素自动水平居中
- ✅ 跨浏览器兼容性好
- ✅ 响应式友好

### 问题2：颜色选择说明

**为什么选择 #1f2937？**
- 深灰黑色，专业度高
- 在红色背景上对比度好
- 符合现代设计趋势
- 比纯黑色（#000）更柔和

**为什么英文名用 #4b5563？**
- 比中文名颜色稍浅
- 形成视觉层次
- 保持可读性

### 问题3：HTML元素移动技术

**使用BeautifulSoup**：
```python
# 1. 查找案例section
case_section = soup.find('section', text=lambda t: '香港中小企' in t if t else False)

# 2. 查找FAQ section
faq_section = soup.find('section', text=lambda t: '常見問題' in t if t else False)

# 3. 移动案例到FAQ之前
case_section_copy = case_section.extract()
faq_section.insert_before(case_section_copy)
```

**优点**：
- ✅ 保留完整HTML结构
- ✅ 不影响样式和脚本
- ✅ 批量处理高效
