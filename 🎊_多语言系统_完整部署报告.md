## 💼 使用方法

### 开发者：内容同步

```bash
# 1. 修改中文版页面
vim dashboard.html

# 2. 同步到其他语言版本（3秒完成）
./sync-multilingual.sh dashboard

# 或一键同步所有页面
./sync-multilingual.sh all

# 3. 完成！✅
# - en/dashboard.html（英文版）已更新
# - jp/dashboard.html（日文版）已更新
# - kr/dashboard.html（韩文版）已更新

# 4. 数据完全同步 ✅

```
