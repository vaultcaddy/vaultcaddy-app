## 🚀 使用方法

```bash
# 同步单个页面
python3 multilingual_sync_master.py dashboard

# 同步所有页面
python3 multilingual_sync_master.py all

# 预览模式（不实际写入）
python3 multilingual_sync_master.py dashboard --dry-run
```

## 💡 工作流程

1. 修改中文版页面（dashboard.html等）
2. 运行同步脚本
3. 自动翻译并创建/更新其他语言版本
4. 验证各语言版本
