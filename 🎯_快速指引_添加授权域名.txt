# 🎯 快速指引：添加授权域名

## 您当前的问题

❌ 在 Google 提供商配置页面找不到授权域名设置

## ✅ 解决方法

**授权域名不在当前页面！**

需要去 **Settings（设置）** 页面

---

## 🚀 3步快速操作

### 步骤1：返回上一级
```
点击页面顶部的 "Authentication" 文字
```

### 步骤2：点击 Settings 标签
```
在顶部标签栏找到并点击 "Settings"（设置）
```

### 步骤3：添加域名
```
向下滚动 → 找到 "Authorized domains"
→ 点击 "Add domain"
→ 输入：vaultcaddy.com
→ 保存
```

---

## 🔗 或直接访问

```
https://console.firebase.google.com/u/1/project/vaultcaddy-production-cbbe2/authentication/settings
```

然后向下滚动找到 "Authorized domains"

---

## 📊 页面结构

```
Authentication
├── 用户（Users）
├── 登录方法（Sign-in method）
│   └── Google 配置  ← 您现在在这里
├── 模板（Templates）
└── 设置（Settings）  ← 授权域名在这里！
    └── Authorized domains  ← 目标位置
```

---

## ✅ 完成后

Google 登录就可以正常工作了！
