## 📊 文案优化核心原则

### 1️⃣ 搜索结果文案公式

**Title（标题）= 核心价值 + 数字 + 价格 + 情感词**

```
❌ 错误示例：VaultCaddy - AI Document Processing
   （平淡无奇，无差异化）

✅ 正确示例：银行对账单转Excel｜3秒完成｜比人工便宜95%｜月费$46起
   （价值明确，数字吸睛，价格透明）
```

**Description（描述）= 痛点 + 解决方案 + 数字证明 + 行动呼吁**

```
❌ 错误示例：Use AI to process bank statements automatically
   （太笼统，无说服力）

✅ 正确示例：每月处理对账单要30小时？VaultCaddy AI帮您3秒搞定，比人工便宜95%，
   比Dext便宜70%。支持所有香港银行，免费试用20页。立即体验！
   （痛点共鸣，数字对比，立即行动）
```

### 2️⃣ 文案心理学技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **对比锚定** | 与昂贵方案对比 | "比Dext便宜70%，比人工便宜95%" |
| **具体数字** | 避免模糊表达 | "3秒"而不是"快速"，"98%"而不是"准确" |
| **社会证明** | 展示用户数量 | "1,000+香港企业使用" |
| **紧迫感** | 限时/限量 | "免费试用20页（今天注册立即获得）" |
| **降低门槛** | 消除顾虑 | "无需信用卡，3秒看到效果" |
| **本地化** | 地域相关性 | "支持所有香港银行""专为香港企业设计" |

### 3️⃣ 情感触发词清单

**正面情感**（让用户想要）：
- 快速、简单、轻松、自动、智能、专业
- 节省、便宜、实惠、超值、免费
- 安全、可靠、准确、精准、保证

**负面情感**（痛点共鸣）：
- 繁琐、耗时、昂贵、复杂、容易出错
- 加班、熬夜、返工、压力大

**行动词**（激发行动）：
- 立即、马上、现在、今天
- 免费、试用、体验、开始
- 节省、提升、告别、解放
