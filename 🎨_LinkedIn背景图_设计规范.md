## 📝 Canva完整操作指南

### 准备工作

**需要的素材**:
```
1. VaultCaddy Logo (透明背景PNG)
2. 产品截图（可选）
3. 银行Logo（如使用方案C）
4. 客户Logo（如使用方案C，需获得许可）
```

---

### 详细步骤（方案A）

#### Step 1: 创建项目
```
1. 访问 canva.com
2. 点击"Create a design"
3. 选择"Custom size"
4. 输入：1584 × 396 px
5. 点击"Create new design"
```

#### Step 2: 背景渐变
```
1. 点击左侧"Background"
2. 选择"Gradient"
3. 点击颜色选择器：
   - 左色块：输入 #667eea
   - 右色块：输入 #764ba2
4. 调整方向：135度（斜向）
```

#### Step 3: 添加网格图案（可选）
```
1. 搜索"grid pattern"
2. 拖动到画布
3. 调整透明度：10%
4. 颜色改为白色
5. 混合模式：Overlay
```

#### Step 4: 添加Logo
```
1. 点击"Uploads"
2. 上传VaultCaddy Logo
3. 拖动到左上角
4. 调整尺寸：80 × 80 px
5. 位置：距离左边缘 40px，距离顶部 40px
```

#### Step 5: 添加主文字
```
1. 点击"Text" > "Add a heading"
2. 输入"VaultCaddy"
3. 字体：Poppins Bold（或 Montserrat Bold）
4. 大小：48px
5. 颜色：#ffffff
6. 位置：Logo右侧
```

#### Step 6: 添加副标题
```
1. 点击"Text" > "Add a subheading"
2. 输入"AI-Powered Bank Statement Automation Platform"
3. 字体：Poppins Regular
4. 大小：20px
5. 颜色：#f0f0f0
6. 位置：主标题正下方
```

#### Step 7: 添加数据卡片（右侧）
```
为每个数据创建一个卡片：

1. 添加矩形：
   - Elements > Shapes > Rectangle
   - 颜色：白色
   - 透明度：15%
   - 圆角：8px
   - 尺寸：约 250 × 50 px

2. 添加文字：
   - "📊 500+ Businesses"
   - 字体：Poppins SemiBold
   - 大小：18px
   - 颜色：#ffffff

重复3次创建3个卡片，垂直排列
```

#### Step 8: 添加底部CTA
```
左侧网站：
- 文字："https://vaultcaddy.com"
- 字体：Poppins Medium，16px
- 颜色：#ffffff

右侧优惠：
- 文字："🎁 20 Pages Free Trial"
- 字体：Poppins Bold，16px
- 颜色：#ffd700（金黄）
```

#### Step 9: 最终调整
```
1. 确保所有元素对齐
2. 检查安全区域（不要太靠边）
3. 预览效果
4. 下载为PNG（推荐）或JPG
```

## 📱 移动端预览

**重要提醒**: LinkedIn在移动端会裁剪背景图

```
移动端显示区域：
- 宽度：全宽（1584px）
- 高度：约 250px（中间部分）
- 顶部和底部可能被裁剪

设计建议：
✅ 关键信息放在中间 250px 高度内
✅ 重要CTA放在中央
❌ 不要在顶部/底部放关键信息
```
