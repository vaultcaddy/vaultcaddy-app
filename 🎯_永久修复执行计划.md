## 🚨 常见问题解决

### Q1: FFmpeg未安装

```bash
# 安装FFmpeg
brew install ffmpeg

# 验证安装
ffmpeg -version
```

---

### Q2: 录制时文件太大

```
不用担心！
- MOV文件可能100MB+
- 脚本会自动压缩到<2MB
- 这是正常的
```

---

### Q3: GIF文件仍然>2MB

```bash
# 手动降低质量
# 编辑 video/convert_videos_to_gifs.sh
# 修改 FPS=20 为 FPS=15
# 修改 QUALITY=80 为 QUALITY=70
# 重新运行转换脚本
```

---

### Q4: 录制出错需要重新录制

```bash
# 直接覆盖原文件即可
# 重新录制并保存为相同文件名
# 然后重新运行转换脚本
```

---

### Q5: 部署后发现问题

```bash
# 恢复备份
find . -name "*.backup_gif_deploy" -exec sh -c 'mv "$1" "${1%.backup_gif_deploy}"' _ {} \;

# 修复问题后重新部署
python3 deploy_localized_gifs.py
```
