모든 기사가 다음을 포함해야 합니다:

### 기본 SEO
- [ ] 타겟 키워드가 Title에 포함
- [ ] 메타 설명 150-160자
- [ ] H1 태그는 하나만
- [ ] H2/H3 계층 구조 명확
- [ ] 이미지 alt 태그
- [ ] 내부 링크 3-5개
- [ ] 외부 링크 (권위 있는 소스)

### Schema 마크업
- [ ] Article Schema
- [ ] FAQ Schema (3개 Q&A 최소)
- [ ] Open Graph 태그
- [ ] 게시일/수정일

### 한국 최적화
- [ ] ₩7,998/월 정확한 가격
- [ ] 한국 12개 은행 언급
- [ ] 한국 도시 사례 연구
- [ ] 한국 회계 규정 준수

### 모바일 최적화
- [ ] 반응형 디자인
- [ ] 빠른 로딩 속도
- [ ] 읽기 쉬운 글꼴 크기
