## 🛠️ 方案B: 使用Figma/Canva（专业）⭐⭐⭐⭐

### Figma (免费)

**网址**: https://www.figma.com/

**步骤**:

```
1. 导入GIF每一帧为图片
2. 在每帧上添加文字层
3. 遮盖原文字（白色矩形）
4. 添加新文字（对应语言）
5. 导出为PNG序列
6. 使用在线工具合并为GIF
```

---

### Canva (免费)

**网址**: https://www.canva.com/

**步骤**:

```
1. 上传GIF
2. 使用"元素"添加形状遮盖原文字
3. 添加文本框，输入新文字
4. 调整字体、大小、颜色
5. 下载为GIF
```
