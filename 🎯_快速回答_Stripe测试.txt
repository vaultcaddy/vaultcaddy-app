## 🚀 如何修改为月付

**Stripe Dashboard：**
```
1. 登录：https://dashboard.stripe.com/
2. Products → Payment links
3. 找到年付链接
4. Edit → Billing period: Yearly → Monthly
5. Amount: HK$744 → HK$62
6. 保存
```

**或创建新链接：**
```
1. 创建新产品：VaultCaddy 年付方案（月结）
2. 价格：HK$62/月
3. Metadata: monthly_credits: 100
4. 创建 Payment Link
5. 替换 billing.html 中的链接
```
