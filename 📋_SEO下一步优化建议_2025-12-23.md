### 3. Core Web Vitals优化 ⭐⭐⭐⭐

**目标**: 所有指标达到"Good"级别

**需要优化的指标**:

#### LCP (Largest Contentful Paint)
- 目标: < 2.5s
- 优化方案:
  - 优化Hero图片加载
  - 使用CDN加速
  - 预加载关键资源

#### FID (First Input Delay)
- 目标: < 100ms
- 优化方案:
  - 延迟加载非关键JS
  - 使用Web Workers

#### CLS (Cumulative Layout Shift)
- 目标: < 0.1
- 优化方案:
  - 为图片指定尺寸
  - 为动态内容预留空间
