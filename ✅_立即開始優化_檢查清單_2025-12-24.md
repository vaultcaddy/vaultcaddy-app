**截圖工具**:
- macOS: Command + Shift + 4
- CleanShot X (付費,推薦)
- Skitch (免費,標註)

**圖片壓縮**:
- TinyPNG (線上) - https://tinypng.com
- Squoosh (Google) - https://squoosh.app
- ImageOptim (macOS本地)

**格式轉換**:
```bash
# 安裝cwebp (如果還沒有)
brew install webp

# 轉換為WebP
cwebp -q 85 input.jpg -o output.webp

# 批量轉換
for file in *.jpg; do cwebp -q 85 "$file" -o "${file%.jpg}.webp"; done
```
