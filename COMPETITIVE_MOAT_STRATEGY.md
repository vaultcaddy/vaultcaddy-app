**建立日期**: 2026-04-01
**文件作用**: 記錄 VaultCaddy 在香港市場的競爭優勢與防禦策略。作為總顧問的指導方針，確保團隊資源集中在「能拉開競爭差距」且「能直接轉化為營收」的護城河上。

## 1. 香港市場競爭者分析 (Competitor Landscape)

目前香港中小企處理收據/簿記的主要方式及競爭對手：

| 競爭對手 / 方式 | 運作模式 | 缺點 / 痛點 (我們的機會) |
| :--- | :--- | :--- |
| **傳統會計師樓 (CPA Firms)** | 年底收鞋盒，人工輸入 Excel。 | **極貴** (執數費 $5k-$10k)、慢、老闆無法即時看財務狀況。 |
| **Xero / QuickBooks** | 專業雲端會計軟體。 | **太複雜**。對小老闆來說學習成本極高，員工也不會用。月費貴 ($200-$400+)。 |
| **Dext (前 Receipt Bank)** | 專做收據 OCR，串接 Xero。 | **定位錯配**。他們賣給會計師，不賣給小老闆。英文介面為主，對香港本地單據（如手寫茶餐廳單）辨識率差。 |
| **老闆自己打 Excel** | 每晚對著收據手動輸入。 | **浪費時間**。老闆的時間應該用來做生意，不是做 Admin。 |

## 2. VaultCaddy 的絕對優勢 (Our Unfair Advantages)

1.  **極致的「員工端」零摩擦 (Zero-Friction for Staff)**
    *   *別人的做法*：員工要下載 App、註冊帳號、登入。
    *   *我們的做法*：掃 QR Code -> 拍照 -> 關閉。**這是我們目前最強的賣點**。
2.  **專為「免除執數費」而生的匯出格式 (CPA-Ready Export)**
    *   *別人的做法*：匯出複雜的會計報表。
    *   *我們的做法*：匯出極簡 Excel，**附帶雲端圖片連結**。會計師最愛這種格式，老闆拿著這個 Excel 就能理直氣壯拒絕付「執數費」。
3.  **香港本地化 AI (HK-Localized AI)**
    *   使用 Qwen-VL Max，對中英夾雜、繁體字、甚至部分手寫的香港收據辨識能力，遠超傳統西方的 OCR 引擎。

## 3. 如何建立護城河？(Building the Moat)

既然我們的目標只有「賺錢」，護城河必須建立在「增加轉換成本」和「網絡效應」上。

### 階段一：產品護城河 (短期 - 鎖定用戶)
*   **歷史數據鎖定 (Data Lock-in)**：當用戶累積了半年的收據和 Excel 在我們這裡，他們轉換到其他平台的成本就會變高。
*   **專屬 QR Code 習慣**：一旦店員習慣了「掃碼拍照」的動作，老闆就不敢輕易換系統，因為重新培訓店員很麻煩。

### 階段二：生態護城河 (中期 - 擴大營收)
*   **會計師反向推薦 (CPA Referral)**：當老闆把我們生成的完美 Excel 交給會計師，會計師發現這省了他們初級員工大把時間。我們應該做一個 **「會計師推薦計畫 (Affiliate Program)」**：會計師推薦客戶用 VaultCaddy，會計師抽佣 20%。這會讓會計師成為我們的免費 Sales！

## 4. 總顧問的殘酷建議 (Harsh Truths for Revenue)

1.  **不要做「會計軟體」**：絕對不要去開發資產負債表 (Balance Sheet)、複式簿記 (Double-entry) 等功能。我們是「收據處理器」，不是 Xero。功能越多，客服成本越高，利潤率越低。
2.  **定價策略要狠**：$78/月 是很好的獲客價。但未來可以推出 $198/月的「Agency/多店鋪版」，讓有多間分店的老闆可以生成多個 QR Code。
3.  **流量即一切**：產品已經 Ready，現在 90% 的時間應該花在 TikTok/IG Reels 短影音（拍老闆半夜入數的慘狀 vs 掃碼 5 秒搞定）和 SEO 上。
