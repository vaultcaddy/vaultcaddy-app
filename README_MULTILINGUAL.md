## 💡 常用操作

### 同步所有页面
```bash
./sync-multilingual.sh all
```

### 只同步单个页面
```bash
./sync-multilingual.sh dashboard
./sync-multilingual.sh account
./sync-multilingual.sh billing
```

### 添加新术语
```bash
# 1. 编辑翻译字典
vim multilingual_sync_master.py

# 2. 在 TRANSLATION_DICT 中添加新术语
'新術語': {
    'en': 'New Term',
    'jp': '新しい用語',
    'kr': '새로운 용어'
}

# 3. 重新同步
./sync-multilingual.sh all
```

### 数据互通

## ⚠️ 重要规则

### ✅ 正确做法
- 只修改中文版（根目录的HTML文件）
- 修改后运行 `./sync-multilingual.sh all`
- 让系统自动处理翻译和同步

### ❌ 错误做法
- ❌ 不要直接修改 `en/`、`jp/`、`kr/` 文件夹中的文件
- ❌ 不要为不同语言版本创建不同的Firebase项目
- ❌ 不要手动删除语言切换器
