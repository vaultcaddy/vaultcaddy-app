## 💡 最佳实践总结

### H1标签撰写公式

```
H1 = 品牌名称 + 目标对象 + 核心功能 + 价值主张
```

**示例：**
```
VaultCaddy + 匯豐銀行 + 對帳單AI處理 + 3秒轉QuickBooks | 98%準確率
```

### H2-H6使用规则

1. **H2：主要章节**
   - 功能介绍
   - 定价方案
   - 客户评价
   - 常見問題

2. **H3：子章节**
   - 具体功能点
   - 定价选项
   - 个别评价
   - 单个问题

3. **H4：细节**
   - 用户名称
   - 页脚栏目标题

### 避免的错误 ❌

1. **多个H1标签**
   - 每页只能有1个H1
   - 不要用H1作为标题样式

2. **跳级使用**
   - 不要H1直接跳到H3
   - 必须按顺序：H1 → H2 → H3

3. **缺少H标签**
   - 每页至少要有：1个H1、2个H2
   - 长页面应该有更多H2-H3

4. **H1内容过长**
   - 理想长度：50-70字符
   - 包含最重要的关键词

5. **H1内容过短**
   - 避免只有品牌名
   - 至少包含2-3个关键词
