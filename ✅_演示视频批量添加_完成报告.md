## 🎬 视频播放交互

### JavaScript 功能

```javascript
function playDemoVideo(overlay) {
    const video = overlay.previousElementSibling;
    if (video && video.tagName === 'VIDEO') {
        video.play();           // 开始播放
        overlay.style.opacity = '0';  // 淡出覆盖层
        setTimeout(() => {
            overlay.style.display = 'none'; // 移除覆盖层
        }, 300);
    }
}
```

**用户体验**:
1. 页面加载：显示自定义播放按钮
2. 用户点击：播放按钮淡出，视频自动播放
3. 视频结束：显示原生控制条（可重播、暂停等）
