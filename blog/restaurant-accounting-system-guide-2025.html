2. 餐廳成本控制黃金法則

香港餐飲業有個「黃金公式」，成功餐廳都遵循這個比例：

🎯 餐廳成本黃金比例

食材成本 30-35% 超過40%需警惕
人工成本 25-30% 包含薪金+強積金
租金成本 10-15% 舖租+管理費
其他成本 10-15% 水電+雜費
淨利潤 15-20% 目標利潤

⚠️ 警示：如果利潤<10%，代表成本失控，需立即調整！

食材成本控制5大技巧

技巧1：每日盤點高價食材

海鮮、牛肉、紅酒等高價食材，每日盤點庫存：

  • ✓ 記錄：期初庫存 + 今日進貨 - 今日使用 = 期末庫存
  • ✓ 如果數量不符，代表有浪費或偷竊
  • ✓ 使用VaultCaddy記錄所有進貨單，自動計算庫存

技巧2：計算每道菜成本

製作標準食譜（Standard Recipe），計算每道菜的食材成本：

例子：牛肉炒飯

牛肉150g $18

蔬菜+調味料 $4

售價 $78

目標：成本率30-35%為合理，超過40%需調整定價或減少份量。

技巧3：季節性調整菜單

根據食材價格季節性波動，調整菜單：

  • ✓ 夏季：蔬菜便宜，推廣沙拉、涼拌菜
  • ✓ 冬季：根莖類便宜，推廣湯品、燉菜
  • ✓ 淡季：推出「特價套餐」清理庫存
  • ✓ 旺季：提高招牌菜價格，增加利潤

技巧4：與供應商議價

香港餐飲業供應商競爭激烈，老板有議價空間：

  • ✓ 每月比較3家供應商價格（使用VaultCaddy自動分析）

  • ✓ 直接向批發市場採購（如長沙灣蔬菜批發市場）

技巧5：減少食材浪費

香港餐廳平均浪費10-15%食材，相當於每年損失$100,000-$300,000！

減少浪費方法：

  • ✓ FIFO（First In First Out）：先進先出，避免食材過期

6. 餐廳報稅完整流程

香港餐廳需要處理以下稅務事項：

1. 利得稅（Profits Tax）

稅率：首$2,000,000利潤 → 8.25%，超過部分 → 16.5%

報稅時間：每年4月收到報稅表，5月底前提交

需要文件：

  • ✓ 所有收入記錄（現金+信用卡+電子支付）
  • ✓ 所有支出單據（供應商發票、租金、水電、薪金等）
  • ✓ 銀行對帳單
  • ✓ 審計報告（營業額>$1,000,000需審計）

VaultCaddy幫助：自動整理所有支出單據，一鍵生成Excel報表給會計師。

2. 薪俸稅（Salaries Tax）

義務：僱主需為員工填報IR56B表格

報稅時間：每年4月1日前提交上一年度員工薪金資料

需要文件：

  • ✓ 員工薪金記錄
  • ✓ 強積金供款記錄
  • ✓ 員工入職/離職日期

3. 強制性公積金（MPF）

供款率：僱主5% + 員工5%（月薪$7,100-$30,000）

截數日：每月10號前供款上月MPF

常見報稅錯誤（避免罰款）

  1. 漏報現金收入：稅務局會根據行業平均利潤率估算，如果你的利潤率過低（<10%），會被懷疑逃稅。
  2. 單據保留不足：香港法例要求保留所有單據7年，否則罰款$100,000。
  3. 虛報支出：部分餐廳老板用假發票虛報支出，一旦被查會面臨刑事起訴。
  4. 遲交報稅表：罰款$1,200起，嚴重者可監禁。

Q2: 餐廳食材成本控制在多少才合理？
A: 香港餐飲業標準：食材成本應佔營業額的30-35%。

計算公式：
食材成本率 = (期初庫存 + 本期進貨 - 期末庫存) ÷ 營業額 × 100%
判斷標準：
• 25-30%：優秀（高利潤）
• 30-35%：合理（行業標準）
• 35-40%：需改善（利潤較低）
• 40%+：警報（可能虧損）

如果超過40%，代表：
❌ 定價過低（需調整菜單價格）
❌ 浪費嚴重（需追踪廚餘）
❌ 進貨成本過高（需更換供應商）
❌ 員工偷竊（需加強庫存管理）

控制技巧：
1. 每日盤點高價食材（海鮮、牛肉等）
2. 追踪每道菜成本，淘汰「賠本菜」
3. 季節性調整菜單（夏季推蔬菜、冬季推燉菜）
4. 與供應商議價（比較3家報價）
5. 減少浪費（FIFO原則、標準份量）
