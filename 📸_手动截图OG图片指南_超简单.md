# 📸 手动截图 OG 图片指南 - 超简单版

**目标**：为每个 landing page 创建 Open Graph 预览图  
**方法**：直接截图 Hero 部分  
**时间**：每张 2-3 分钟，总共约 1-2 小时

---

## 🎯 为什么用截图？

✅ **快速** - 无需设计，直接截图  
✅ **真实** - 展示实际页面效果  
✅ **一致** - 与页面风格100%匹配  
✅ **易更新** - 页面改了重新截图即可

## 🛠️ 方法一：使用 Chrome 截图（推荐！）

### 步骤：

#### 1. 打开页面
```
访问：https://vaultcaddy.com/ai-vs-manual-comparison.html
或本地：http://localhost:8000/ai-vs-manual-comparison.html
```

#### 2. 打开开发者工具
```
Mac: Cmd + Option + I
Windows: F12 或 Ctrl + Shift + I
```

#### 3. 切换到移动设备模式
```
点击工具栏的 "手机/平板" 图标
或按快捷键：
Mac: Cmd + Shift + M
Windows: Ctrl + Shift + M
```

#### 4. 设置视口尺寸
```
在顶部输入：
Width: 1200 px
Height: 630 px
```

#### 5. 调整页面显示
```
1. 关闭侧边栏（如有）
2. 滚动到 Hero 部分顶部
3. 确保 Hero 部分完整显示在视口中
4. 可能需要缩放（Zoom: 100%）
```

#### 6. 截图
```
方法 A - 使用 Chrome 截图工具：
1. 按 Cmd + Shift + P (Mac) 或 Ctrl + Shift + P (Windows)
2. 输入 "screenshot"
3. 选择 "Capture screenshot"
4. 图片自动保存到下载文件夹

方法 B - 右键截图：
1. 右键点击页面空白处
2. 选择 "截图" 或 "Capture Screenshot"
```

#### 7. 处理图片
```
1. 找到下载的图片（通常在 ~/Downloads/）
2. 重命名为：ai-vs-manual-comparison.jpg
3. 检查尺寸（应该是 1200x630）
4. 如果尺寸不对，使用下面的调整方法
```

## 🖼️ 方法二：使用 Mac 截图工具

### 步骤：

#### 1. 打开页面
```
全屏浏览器，访问页面
滚动到 Hero 部分
```

#### 2. 截图
```
按 Cmd + Shift + 4（部分截图）
或 Cmd + Shift + 3（全屏截图）
```

#### 3. 框选 Hero 部分
```
按住鼠标左键，拖动框选 Hero 区域
包含：
- 标题
- 4个核心卖点卡片
- CTA 按钮
- 不包含导航栏和页脚
```

#### 4. 调整尺寸
```
使用"预览"应用：
1. 双击打开截图
2. 工具 → 调整大小
3. 宽度：1200 px
4. 高度：630 px
5. 取消勾选"缩放比例"
6. 点击"好"
7. 另存为 JPG（质量 80%）
```

## 📏 图片调整工具

### 如果截图尺寸不对，使用这些工具：

#### 1. **Canva** (最简单！)
```
网址：https://www.canva.com/
步骤：
1. 创建自定义尺寸：1200 x 630 px
2. 上传截图
3. 调整到画布大小
4. 下载为 JPG（质量 80%）
```

#### 2. **Photopea** (免费在线 Photoshop)
```
网址：https://www.photopea.com/
步骤：
1. 打开图片
2. Image → Image Size
3. 设置 Width: 1200, Height: 630
4. Method: Bicubic (best for smooth gradients)
5. File → Export As → JPG (Quality 80)
```

#### 3. **Squoosh** (Google 压缩工具)
```
网址：https://squoosh.app/
步骤：
1. 上传图片
2. Resize: 1200 x 630
3. Format: MozJPEG
4. Quality: 80
5. 下载
```

## 💡 常见问题

### Q1: 截图模糊怎么办？

**答**：
1. 确保浏览器缩放为 100%
2. 使用 Retina/高分辨率显示器
3. 截图后使用 Photopea 锐化
4. 或使用更高分辨率截图（2400x1260），然后缩小

### Q2: 文件太大（> 1MB）怎么办？

**答**：使用 Squoosh.app 压缩：
1. 上传图片
2. Format: MozJPEG
3. Quality: 75-80
4. 下载（通常减少 50-70%）

### Q3: 截图包含了不需要的部分？

**答**：
1. 使用 Canva 重新裁剪
2. 或使用 Photopea 的裁剪工具
3. 确保最终尺寸为 1200x630

### Q4: 不同页面截图风格不一致？

**答**：
1. 使用相同的浏览器窗口尺寸
2. 使用相同的缩放比例（100%）
3. 使用相同的截图工具
4. 使用相同的后期处理（压缩、锐化）
