### 阶段3: 监控和优化（持续）

#### A. 每周监控

**Google Search Console检查**:
1. 查看"查询"报告，找出有曝光的关键词
2. 记录曝光次数和点击次数
3. 计算CTR（点击率）
4. 识别高曝光但低CTR的关键词

**关键指标**:
- 曝光次数 > 10次 → 值得优化
- CTR < 2% → 需要优化Title/Description
- 排名在20-50位 → 可以深度优化提升排名
- 排名在50位之后 → 需要更多内容/外链

#### B. 优化策略

**高曝光低点击（CTR < 2%）**:
1. 优化Title：添加评分、CTA、更吸引人的描述
2. 优化Description：突出关键卖点、免费试用
3. 添加Rich Snippets：评分、FAQ、HowTo
4. 优化页面内容：确保与搜索意图匹配

**高曝光有排名但无点击（排名10-20位）**:
1. 深度优化Title/Description
2. 添加更多内链
3. 获取外链
4. 优化页面内容深度

**高曝光高排名但无点击（排名1-10位）**:
1. 可能是搜索意图不匹配
2. 优化页面内容，确保满足用户需求
3. 考虑购买该关键词广告
