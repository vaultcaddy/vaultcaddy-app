### 任务清单

#### 1. Core Web Vitals极致优化（6小时）

**LCP（Largest Contentful Paint）优化**
- [ ] 转换所有图片为WebP格式
  ```bash
  # 批量转换
  for img in $(find . -name "*.png" -o -name "*.jpg"); do
      cwebp -q 85 "$img" -o "${img%.*}.webp"
  done
  ```
- [ ] 实现响应式图片
  ```html
  <picture>
    <source srcset="image-480w.webp 480w, image-800w.webp 800w" type="image/webp">
    <img src="image.png" alt="描述" loading="lazy">
  </picture>
  ```
- [ ] 优化字体加载
  ```html
  <link rel="preload" href="fonts/main.woff2" as="font" type="font/woff2" crossorigin>
  ```

**FID（First Input Delay）优化**
- [ ] 代码分割（Code Splitting）
- [ ] 移除未使用的JavaScript
- [ ] 使用Web Workers处理耗时任务

**CLS（Cumulative Layout Shift）优化**
- [ ] 为所有图片设置width和height
- [ ] 为动态内容预留空间
- [ ] 避免在现有内容上方插入内容

**目标指标**:
