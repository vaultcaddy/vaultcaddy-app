## 🚀 技术实现

### FFmpeg转换命令

**步骤1: 生成调色板**
```bash
ffmpeg -i "Chase Bank vaultcaddy.mp4" \
  -vf "fps=10,scale=900:-1:flags=lanczos,palettegen" \
  -t 15 \
  palette.png
```

**步骤2: 使用调色板生成GIF**
```bash
ffmpeg -i "Chase Bank vaultcaddy.mp4" -i palette.png \
  -lavfi "fps=10,scale=900:-1:flags=lanczos [x]; [x][1:v] paletteuse" \
  -t 15 \
  chase-bank-demo.gif
```

**结果**:
- ✅ 高质量256色调色板
- ✅ Lanczos缩放算法（最佳质量）
- ✅ 优化的抖动处理
- ✅ 文件大小仅1.1MB
