## 🎬 制作规范

### 视频标准

#### 画质
- 分辨率：1920x1080 (Full HD)
- 帧率：30fps
- 格式：MP4 (H.264)

#### 音频
- 采样率：48kHz
- 比特率：192kbps
- 格式：AAC
- 使用专业麦克风

#### 字幕
- 中文字幕（必须）
- 英文字幕（可选）
- 使用YouTube自动生成后人工校对

### 视频结构模板

```
00:00-00:05  品牌片头动画
             "VaultCaddy 會計自動化教室"
             
00:05-00:30  开场（Hook）
             - 提出问题或痛点
             - 吸引观众继续看
             
00:30-XX:XX  主要内容
             - 清晰的章节划分
             - 每3-5分钟一个小节
             
XX:XX-XX:XX  总结和要点回顾
             - 重申3-5个关键点
             
XX:XX-XX:XX  CTA（Call to Action）
             - 免费试用链接
             - 订阅频道
             - 相关视频推荐
```

### 视觉元素

1. **片头**
   - 5秒品牌动画
   - Logo + 口号

2. **标题卡**
   - 每个章节标题
   - 统一设计风格

3. **图表动画**
   - 数据可视化
   - 对比表格
   - 流程图

4. **屏幕录制**
   - 高清录制
   - 添加鼠标高亮
   - 关键操作慢动作

5. **字幕**
   - 黄色底色，黑色文字
   - 字号适中，易读

## 📊 SEO优化策略

### 标题优化

**公式**: [数字] + [关键词] + [价值主张] + [年份]

**示例**:
- ❌ "VaultCaddy教程"（太笼统）
- ✅ "VaultCaddy 3分鐘快速上手 | 銀行對帳單AI處理 | 2025最新教程"

**原则**:
- 前60个字符最重要
- 包含1-2个主要关键词
- 吸引点击的词汇（快速、完整、免费等）

### 描述优化

**结构**:
```
[第1段] 视频内容概述（2-3句，包含关键词）

[第2段] 详细目录（带时间戳）
00:00 - 開場
01:30 - 功能演示
03:45 - 實際案例
...

[第3段] 相关资源链接
🔗 VaultCaddy官网：https://vaultcaddy.com
📚 完整教程文章：https://...
💬 加入社群：...

[第4段] 频道介绍和订阅CTA

[第5段] 关键词标签
#VaultCaddy #銀行對帳單 #AI會計 #QuickBooks #會計自動化
```

### 标签（Tags）策略

**每个视频20-25个标签**

**标签类型**:
1. **品牌标签**（2个）
   - VaultCaddy
   - VaultCaddy教程

2. **核心关键词**（5-8个）
   - 銀行對帳單處理
   - AI會計
   - QuickBooks整合
   - 對帳單轉Excel
   - 會計自動化

3. **长尾关键词**（8-10个）
   - 匯豐銀行對帳單
   - QuickBooks香港
   - PDF轉Excel
   - 手機拍照記帳

4. **竞品关键词**（2-3个）
   - QuickBooks教程
   - Xero教程
   - 會計軟件

5. **通用关键词**（2-3个）
   - 會計教程
   - 財務管理
   - 小企業記帳

### 缩略图设计

**标准规格**:
- 尺寸：1280x720 px
- 格式：JPG或PNG
- 文件大小：<2MB

**设计原则**:
1. **高对比度**
   - 明亮背景 + 深色文字
   - 或相反

2. **大号文字**
   - 3-5个字最佳
   - 可在手机上清晰阅读

3. **人脸/表情**
   - 真人出镜更吸引点击
   - 表情要生动

4. **品牌元素**
   - Logo
   - 统一的配色方案

```

[中央：大号白色文字] "3秒完成"
[右上角：VaultCaddy Logo]
[左下角：白色小字] "AI自動處理"
```
