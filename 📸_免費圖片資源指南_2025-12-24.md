## 🔧 Unsplash URL參數說明

我已經為所有圖片添加了優化參數：

```
?w=1920&h=800&fit=crop
```

**參數解釋**:
- `w=1920` - 寬度1920像素
- `h=800` - 高度800像素  
- `fit=crop` - 裁切以適應尺寸

**自動優化特性**:
- 根據用戶設備自動調整尺寸
- 自動轉換為WebP格式(支援的瀏覽器)
- 自動壓縮以減少文件大小

## 🎨 圖片優化建議

### 如果您想使用自己的圖片

**步驟1: 壓縮圖片**
```bash
# 使用線上工具
1. TinyPNG - https://tinypng.com
   - 上傳圖片
   - 自動壓縮
   - 下載壓縮後的圖片

2. Squoosh - https://squoosh.app
   - Google出品
   - 支援WebP轉換
   - 可對比壓縮效果
```

**步驟2: 轉換為WebP格式** (可選，提升載入速度50%)
```bash
# 如果您有Homebrew (macOS)
brew install webp

# 轉換圖片
cwebp -q 85 input.jpg -o output.webp
```

**步驟3: 上傳並替換**
```html
<!-- 使用picture標籤提供備用格式 -->
<picture>
    <source srcset="your-image.webp" type="image/webp">
    <img src="your-image.jpg" alt="描述" loading="lazy">
</picture>
```

## 💡 關於銀行Logo的說明

### 為什麼使用文字而非實際Logo?

**原因**:
1. **版權問題** - 銀行Logo有版權，未經授權使用可能違法
2. **簡化流程** - 不需要下載和管理Logo文件
3. **快速載入** - 文字比圖片載入更快
4. **可訪問性** - 文字對屏幕閱讀器友好

**當前方案**:
```html
<div class="bank-logo" style="display: inline-block;">
    <strong style="color: #DB0011; font-size: 1.8rem;">
        HSBC 匯豐銀行

</div>
```

**優點**:
- ✅ 清晰易讀
- ✅ 使用銀行品牌色

- ✅ SEO友好
