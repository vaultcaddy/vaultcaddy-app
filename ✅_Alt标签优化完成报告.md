## 💡 Alt标签最佳实践

### ✅ 应该做的

1. **描述图片内容**
   - 说明图片显示的是什么
   - 提供上下文信息
   - 包含相关关键词

2. **长度适中**
   - 理想长度：50-125字符
   - 简洁但描述性强
   - 包含最重要的信息

3. **独特且具体**
   - 每张图片的Alt都不同
   - 避免使用通用描述
   - 与页面内容相关

4. **自然融入关键词**
   - 关键词自然出现
   - 不影响可读性
   - 符合用户体验

### ❌ 避免的错误

1. **关键词堆砌**
   ```html
   ❌ alt="VaultCaddy VaultCaddy 銀行對帳單 對帳單 QuickBooks QuickBooks"
   ```

2. **使用冗余词**
   ```html
   ❌ alt="一張照片顯示VaultCaddy用戶..."
   ❌ alt="這個圖片是..."
   ```

3. **Alt标签过短**
   ```html
   ❌ alt="用戶"
   ❌ alt="John"
   ```

4. **Alt标签过长**
   ```html
   ❌ alt="VaultCaddy是一個專門為香港會計師設計的銀行對帳單AI處理工具，可以自動識別和分類銀行交易記錄，並且可以一鍵匯出到QuickBooks、Xero、Excel等會計軟件，節省90%的手動輸入時間..."
   ```

5. **缺少Alt或留空**
   ```html
   ❌ <img src="user.jpg">
   ❌ <img src="user.jpg" alt="">
   ```

### 📝 Alt标签撰写公式

```
Alt = [品牌名] + [主体] + [职业/类型] + [行动/场景]
```

**示例：**
```
VaultCaddy用戶 + John M. + 香港會計師 + 使用VaultCaddy處理銀行對帳單
```
