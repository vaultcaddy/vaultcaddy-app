## 🎯 差異化策略核心

### 避免關鍵詞競食

**問題**: 之前所有銀行頁面Title都太相似
```
❌ 恒生銀行對賬單+收據+發票轉Excel｜3秒處理｜...
❌ 中銀香港對賬單+收據+發票轉Excel｜3秒處理｜...
   ↑ Google會認為是重複內容，互相競爭
```

**解決方案**: 每個銀行有獨特的關鍵詞組合
```
✅ 恒生 → 月結單 + 教學 + 中小企
✅ 中銀 → 多幣種 + iBanking + 官方背景
✅ 渣打 → OCR + 外資 + 國際業務
✅ 東亞 → Cyberbanking + 本地 + 老牌
✅ 花旗 → 信用卡 + 美資 + Corporate Card
```
