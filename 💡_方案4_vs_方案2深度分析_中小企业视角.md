### 问题4: 价格锚定失效 ⚓

**定价心理学**:

**方案2（单一定价）**:
```
用户看到: $2.88/月（100页）

心理锚点: $2.88
参考对象: 无
价值感知: "便宜"

但是！
当用户处理200页:
- 实际付费: $6.78
- 心理感受: "怎么变这么贵？"
- 价值感知: 从"便宜"变成"不确定"
```

**方案4（双层定价）**:
```
用户看到: 
- Starter $2.88/月
- Pro $14.99/月

心理锚点: $14.99（高价锚点）
参考对象: Starter vs Pro
价值感知: "Starter真便宜"、"Pro好像也不贵"

当用户处理200页:
选择Starter:
- 实际付费: $6.78
- 对比Pro: "还是比$14.99便宜"
- 价值感知: "还行"

选择Pro:
- 实际付费: $14.99
- 对比Starter超额: "比$6.78贵一点，但无限"
- 对比竞品: "比$19.99便宜25%"
- 价值感知: "超值"
```

**定价心理学原理**:
```
单一定价 = 没有对比 = 用户只能对比"自己的预期"
双层定价 = 有内部对比 = 用户对比"两个选项"

研究显示: 
- 有对比的定价，转化率高 30-40%
- 高价锚点让低价看起来更便宜
- 三个选项比两个好，两个比一个好
```
